# (🐱 || 🐶 || 😯) 키우기 게임

### 기능 명세서
#### CLI를 통해 게임을 시작한다

- CLI가 지원하는 기능
    - status
        - OO이들의 이름, 포만감, 친밀도를 출력한다
    - create(name)
        - OO이를 새로 만든다
        - create를 치면 '이름을 지어주세요 : ' 라고 뜬다
        - 이름을 작성하면 OO이가 만들어진다
    - sayHi(name)
        - sayHi '대상' -> 인사함
        - sayHi 대상은 친밀도 +1 
    - feed(name)
        - 포만감이 랜덤으로 올라감
        - 시간에 따른 포만감 감소 (명령어 칠 때 마다)
    - save()
        - 현재 있는 OO들의 정보를 JSON 으로 저장한다
